ssh
===

.. warning::

    This formula is deprecated. Please use https://github.com/saltstack-formulas/openssh-formula instead.

.. note::

Available states
================

.. contents::
    :local:

``ssh.client``
--------------

    Installs the ssh client package.

``ssh.server``
--------------
    Installs the ssh server package and enable the service.
